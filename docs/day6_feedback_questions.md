# Day6 初回フィードバック質問項目

GitHub Issueではなく、Google Formで作成する前提の質問項目(最大6問)。初期10人に触ってもらった後、この項目で感想を聞く。

Google Formでの作成手順: forms.google.com → 空白のフォーム → 以下を順に入力。所要時間は3分程度の想定。

---

### Q1. 何に使いましたか?(記述式・任意)

- 種類: 段落(長文回答)
- 必須: 任意
- プレースホルダー例: 「地方自治体向け提案書の構成案を比較した」など

### Q2. 使いづらかった点は何ですか?(記述式・任意)

- 種類: 段落(長文回答)
- 必須: 任意

### Q3. 良かった点は何ですか?(記述式・任意)

- 種類: 段落(長文回答)
- 必須: 任意

### Q4. また使いますか?(選択式・必須)

- 種類: ラジオボタン
- 選択肢: 「また使いたい」「たぶん使わない」「わからない」
- 必須: 必須

### Q5. 代わりに使っているツールはありますか?(記述式・任意)

- 種類: 記述式(短文)
- 必須: 任意
- プレースホルダー例: 「メモ帳」「Notion」「特に何も使っていない」など

### Q6. その他、自由にご意見をください(記述式・任意)

- 種類: 段落(長文回答)
- 必須: 任意

---

## 運用メモ

- 回答は匿名可(メールアドレス収集は行わない)
- フォームのリンクは告知文([day6_announcement_drafts.md](./day6_announcement_drafts.md))と個別案内([issue-001_launch_plan.md](./issue-001_launch_plan.md))の両方に添付する
- Kill Criteria判定([issue-001_validation_plan.md](./issue-001_validation_plan.md))に関わる声(「面倒」「Notionで十分」等)は特にQ2・Q5・Q6に出やすいため、回収後は最優先で確認する
